Commentator.

This is a code and comment analysis script that is designed to encourage developers to more thoroughly comment code.

Right now it's fairly simple. It accepts Javascript / CoffeeScript files as input, but eventually should be file type agnostic (comment styles will just need to be defined)

To call the script, call enforce.py and pass in a file name (either relative path or absolute). e.g.

$ python enforce.py test.js
